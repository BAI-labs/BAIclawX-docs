# Telegram

## Prerequisites

- Telegram account

## Configuration Steps

### 1. Create a Bot and Get Bot Token

1. Open Telegram, search for and open **@BotFather** (with verification badge)
2. Send `/newbot`
3. Follow the prompts to enter and create a bot (**must end with `bot`**, e.g., `myclaw123bot`)
4. On success, copy the **Bot Token** (format: `123456:ABCdef...`) and paste it into BAIclaw's **Bot Token** input box

![](assets/Pasted%20image%2020260319003834.png)

### 2. Get User ID

1. Search for and open **@userinfobot** and click Start
2. Get your User ID from @userinfobot and paste it into BAIclaw's **Allowed User IDs** input box
3. Search for the bot you just created in Telegram and start a conversation

# WeCom

## Prerequisites

- WeCom client (must be updated to the latest version)

## Configuration Steps

### 1. Create a Robot

1. In the client → Workbench, click **Smart Robot** → **Create Robot**, select **API Mode** to create

![](assets/Pasted%20image%2020260319013330.png)

![](assets/Pasted%20image%2020260319013354.png)

2. Select to create with **Long Connection** method, and get **Bot ID** and **Secret**

![](assets/Pasted%20image%2020260319013614.png)

### 2. Add WeCom Bot to BAIclaw

Fill in the obtained **Bot ID** and **Secret** into BAIclaw.

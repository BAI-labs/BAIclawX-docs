# Feishu

## Configuration Steps

1. Open the link [One-Click Create Robot](https://open.feishu.cn/page/openclaw?form=multiAgent)
2. Copy the **ID** and **Secret** to BAIclaw's configuration page to complete the setup
3. For more information, refer to: [OpenClaw Feishu Official Plugin Usage Guide (Public Version)](https://bytedance.larkoffice.com/docx/MFK7dDFLFoVlOGxWCv5cTXKmnMh)

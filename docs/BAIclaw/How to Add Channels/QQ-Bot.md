# QQ Bot

## Prerequisites

- QQ account

## Configuration Steps

### 1. Create a Robot

1. Go to [Quick Registration Create QQ Robot](https://q.qq.com/qqbot/openclaw/login.html)
2. Click **Create Robot**
3. On the robot page, find **AppID** and **AppSecret**, click the copy button on the right side of each respectively, and save to notepad or memo
   > **Note:** **AppSecret** does not support plain text storage. After leaving the page, viewing it again will force a reset. Be sure to save it properly.

![](assets/Pasted%20image%2020260319003627.png)

### 2. Add QQ Bot to BAIclaw

Enter **AppID** and **AppSecret**
